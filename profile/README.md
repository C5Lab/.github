# LAB5 — ESP32-C5 Research & Development Group

We are **LAB5** (*Labolatorium*), an independent hardware and software development group focused on the **ESP32-C5** ecosystem. We design and build custom PCBs based on the ESP32-C5 microcontroller and write fully original firmware to go along with them.

Our work spans offensive security tools, embedded applications, and custom hardware — all tightly integrated with the **M5Stack ecosystem**. We push the boundaries of what's possible on ESP32-C5 by building things from scratch rather than relying on off-the-shelf solutions.

## What we build
- Custom ESP32-C5 hardware (PCBs, dev boards, handheld devices)
- Original firmware in C — no shortcuts, no copy-paste
- Security-focused tools: Evil Twin attacks, deauthers, WPA3-SAE overflow, captive portals, password verification
- M5Stack-compatible devices and applications (Cardputer ADV, Tab5)
- Flipper Zero companion apps for C5 boards
